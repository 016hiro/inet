Network route visualizer example

Goals

Visualizing network routes is advantageous, because we can trace a packet's route from the source node toward to the destination node without inspect routing tables. It is especially helpful if there is a dynamic routing protocol in the network. If there is a change in the routes e.g a router go wrong, we get information immediately about that.

The example consists of 3 simulation models, each demonstrating different features of network route visualization.
