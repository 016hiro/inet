Data link visualizer example

Goals

Visualizing data links is useful, because based on this we can determine the second layer links in the network. It is especially useful in wireless networks, because we can see which devices communicate with each other. In wired networks it is also helpful, e.g when a packet doesn't reach the destination node. In this case we can see at a glance where the communication is jammed.
